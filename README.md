# 🛒 Mini E-Commerce Flask App

A full-stack e-commerce web application built using Flask.

## 🚀 Features
- User authentication (Buyer / Seller)
- Product listing
- Add / Update / Delete products (Seller only)
- Order system (Buyer)
- Cart and payment simulation
- Order history
- Session-based login system

## 🧑‍💻 Tech Stack
- Flask
- SQLAlchemy
- SQLite
- HTML / CSS
- Jinja2
- Gunicorn
- Flask-Caching
